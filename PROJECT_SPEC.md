 # 프로젝트 사양

 ## 개요

 목표: 사용자가 폼을 작성하면 제출된 데이터를 바탕으로 A4 크기의 PDF를 생성하는 반응형 마이크로사이트를 구축하는 것입니다. 생성된 PDF는 A4 용지에 인쇄하거나 디지털로 다운로드하기에 적합해야 합니다.

 ## 기능

- 반응형 입력 폼(텍스트, 날짜, 체크박스, 셀렉트, 이미지 등)
- 실시간 미리보기(선택 사항)
- A4 크기(210mm x 297mm) PDF 생성
- 접근성: 시멘틱 HTML, 라벨 연결, 키보드 탐색
- PDF 내보내기/다운로드 및 인쇄 지원

 ## 사용자 스토리

1. 사용자는 폼을 작성하고 "PDF 다운로드"를 클릭하면 입력 내용을 포함한 A4 규격의 PDF 파일을 받을 수 있다.
2. 모바일 사용자는 편리하게 폼을 작성할 수 있으며, 데스크톱과 동일한 데이터로 PDF를 생성할 수 있다.
3. 관리자는 React 컴포넌트를 수정하여 PDF 템플릿 레이아웃을 업데이트할 수 있다.

 ## 비목표(Non-Goals)

- 인증 및 사용자 계정 관리(MVP 범위 제외)
- 대규모 파일 저장소 구축
